# Rychlý start

Za 5 minut budete mít pracující zařízení: ESP32-C3 půjde online a objeví se na portálu.

Budete potřebovat: desku ESP32 (ESP32-C3 DevKit, Super Mini nebo kompatibilní), USB kabel, PlatformIO, WiFi 2.4 GHz a účet na [portal.idryer.org](https://portal.idryer.org/).

- [Začněte za 5 minut](01-five-minutes.md) — postupný průvodce vašim prvním pracujícím příkladem.
- [Podrobné nastavení](99-detailed-setup.md) — úplný rozpis prostředí, dev REPL, řešení problémů před prvním spuštěním.
